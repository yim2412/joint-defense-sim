# 위협 접근 방위 현실화 (v15.03) 설계

> 상태: 설계 합의 완료 (2026-06-12) · 난이도 중간 · 모델 Opus
> 목적: 시나리오·적 편대·해역에 맞는 **지리적 방향**에서 위협이 접근하도록. 현재는 대부분 랜덤.

## 1. 현재 동작
`_build_threats`(engine_v7.py ~1568) 방위 결정:
- 다방위 옵션 ON → 2~4 섹터 분산
- 대한해협 해역 → `STRAIT_BEARING`
- 그 외 전부 → **랜덤** `random.uniform(0,360)`

→ 적 편대(프리셋/랜덤/커스텀) 무관하게 방위 랜덤. 시나리오 골라도 방향 안 맞음.

## 2. 방위 좌표계
`LatLon.from_xy(x,y)`: x=동(경도), y=북(위도). bearing: **동=0°·북=90°·서=180°·남=270°** (atan2(dy,dx)).
위협은 함대(원점=_REGION_REF) 기준 bearing 방향 거리에 스폰 → 함대로 접근.

## 3. 설계: 좌표 기반 자동 방위 (실측, 해역 무관 정확)
적 세력 **실제 발진 좌표**를 두고, 함대 위치에서 bearing 자동 계산. 절대각 하드코딩 대신 좌표라 해역 바뀌어도 정확. [[feedback-real-data]] 부합.

### 적 세력 발진 좌표 `_THREAT_ORIGIN`
| 세력키 | 거점 | 좌표 |
|--------|------|------|
| china | 칭다오(북해함대) | 36.1, 120.3 |
| china_carrier | 대한해협 남측(동해 진입 경유) | 33.5, 129.5 |
| nk_west | 남포(북한 서해) | 38.7, 125.4 |
| nk_east | 원산(북한 동해) | 39.2, 127.4 |
| russia | 블라디보스토크 | 43.1, 131.9 |
| japan | 마이즈루(동해측) | 35.5, 135.4 |

**현실적 경로 보정**: 중국 항모는 칭다오 직선이 아닌 대한해협 경유 북상 → china_carrier(남쪽 진입). 북한 동해는 원산 좌표대로(남서, "북쪽 느낌" 보정 안 함).

### 프리셋 → 세력 (이름 키워드 자동 분류 `_preset_origin`)
- '북한' 포함 → nk (함대 경도<127=nk_west, ≥127=nk_east)
- '러시아' 또는 '쓰시마' 포함 → russia (쓰시마 봉쇄=Tu-22M3·슬라바·킬로=러 자산)
- '항모전단' 포함(랴오닝·산둥·푸젠) → china_carrier
- 그 외(A2AD·항모 킬 체인·수상함·대잠·전면전·중국 3축·입체·이어도·대한해협 통과 등) → china

### 해역 기본 세력 (프리셋 없음 = 랜덤/커스텀 모드) `_REGION_DEFAULT_ORIGIN`
- 서해 → china · 동해 북부 → russia · 동해 중부 → japan · (대한해협은 STRAIT가 처리)

## 4. 방위 결정 우선순위 (높을수록 우선)
1. 다방위 옵션 ON → 섹터 분산 (사용자 의도 존중, 기존)
2. 대한해협 해역 → STRAIT_BEARING (기존)
3. **[신규] 적 세력 방위** (프리셋 origin 또는 해역 기본) → 좌표서 bearing 계산, ±ORIGIN_SPREAD(약 35°) per-threat
4. 폴백 → 랜덤 (origin 해결 불가 시)

## 5. 구현
- `_THREAT_ORIGIN`·`_REGION_DEFAULT_ORIGIN` 좌표/매핑 테이블 (engine_v7 모듈 전역)
- `_preset_origin(name)` 키워드 분류 + `_threat_origin_bearing()` (함대→세력 bearing rad, 북한 서/동 자동)
- `_build_threats` else 분기에 origin 방위 삽입(per-threat ±spread). 기존 1·2·4 유지 → 하위 호환
- `ORIGIN_SPREAD_DEG ≈ 35`

## 6. 회귀·검증
- ⚠️ 위협 방위 랜덤→고정 변경 → 교전 결과(요격률 등) 변동 → **회귀 FAIL 예상 = 의도된 변경**. 새 동작 검증 후 `verify_regression.py --update`.
- 적정성: 방위 고정으로 밸런스 급변 없는지 ON/OFF 통계 비교.
- 3D 전장: 좌표 따라가니 자동 반영(위협이 지리 방향에서 접근).
- exe 스모크: 시나리오별 위협 접근 방향 육안 확인(3D 전장).
