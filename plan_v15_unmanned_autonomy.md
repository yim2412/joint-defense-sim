# v15 미구현 — 무인·자율 자산 구현 계획 (트랙 A→B→C)

> **배경**: v16 종합 감사(2026-06-30) 후 로드맵 점검에서, v15(AI) 블록의 무인·자율 4종
> (v15.3 자율교전·v15.4 적 무인군집·v15.5 정찰드론·v15.6 아군 USV/UUV)이 미구현으로 확인됨.
> v15는 적응형AI·RL·즉시예측만 하고 무인 자산을 통째로 건너뛰고 v16으로 넘어왔다.
> v16.08 연안 자폭드론 스웜·v16.11 자폭 USV로 "적 비대칭 위협"만 부분 부활, 아군 무인 운용은 공백.
>
> **공통 원칙**: 전부 `enable_xxx` 3종세트·기본 OFF·실험적 도입. 단발+전장 공통.
> OFF 시 회귀 bit-identical 보장. 각 트랙 완료 시 ON/OFF 기준값 측정 + 코드 감사.
>
> **순서**: A(아군 ISR, 가치·재활용 최고) → B(적 무인군집, A의 USV로 대응 구도) → C(자율교전, 독립·고난도).

---

## 트랙 A — 아군 무인 ISR·자산 (v15.5 정찰 드론 + v15.6 USV/UUV) · 난이도 중간

탐지 확장은 모든 교전에 영향 → 가치 최고. 기존 항공/함정 탐지 인프라 재활용.

### A-1. 아군 정찰 드론 편대 (v15.5)
- **engine_core.py FRIENDLY_AIRCRAFT_DB**: 고정익 무인정찰기 1~2종 추가(예: 송골매급 전술 UAV / MQ-9 시가디언급 MALE). 탐지센서 강함·무장 없음(ISR 전용) 또는 경무장.
- **효과**: 레이더 수평선 너머(OTH) 표적 탐지·유도 정보 제공 → 함대 탐지거리/조기경보 확장. 기존 `_detect_range_m`·첫탐지 로깅(`_log_detections`) 인프라에 드론 탐지 소스 추가.
- **드론 개별 피격·격추**: 적 항공위협·SAM에 드론 격추 가능(저생존성). 배치 비용 vs 탐지 이득 트레이드오프.
- **토글**: `enable_recon_drone`. 재활용: P-8A/P-3C 초계 패턴·`FriendlyAircraftObj`.

### A-2. 무인 수상정·수중정 (USV·UUV) (v15.6)
- **engine_core.py SHIP_DB**: 아군 무인 수상정(USV)·무인 잠수정(UUV) 추가. `is_unmanned` 플래그(생존성 낮음·저비용·무인이라 인명손실 0).
- **임무 3종**: ①정찰(탐지 전진 배치) ②기뢰 탐색(v16.10 소해 `enable_minesweeping`와 연계 — UUV가 소해 담당) ③자율 교전(경무장 USV가 적 소형정 대응).
- **MUM-T(유무인 협동)**: 유인 함정과 협동 운용, 통신 두절 시 자율 모드 전환(추상화 — 자율 시 판단 단순화).
- **토글**: `enable_unmanned_assets`. 재활용: v16.10 소해 로직·`is_shore_battery` 면제 패턴(USV/UUV도 침수/회피 특수 처리)·v16.11 자폭 USV 이동 로직.

### A 검증
- 회귀 OFF bit-identical. ON 시 탐지거리 확장으로 첫탐지 시각 앞당겨짐·요격률 상승 측정.
- 정찰드론 격추율·USV 손실 ON/OFF 기준값. 기준값 메모리 `project-baseline-unmanned-isr`.

## 트랙 B — 적 무인기 군집 (Swarm) (v15.4) · 난이도 중간~높음

수십~수백 대 분산 경로 동시 접근 → 개별 요격 강제 → 탄약 급소모. 레이저(v17.2)와 짝.

- **⚠ 과거 제거 이력**: v7.20.01에서 "현실성 부족"으로 드론 떼(Swarm-12) 제거됨. **재도입은 현실적 군집 모델이 관건** — 규모·RCS·속도·교전 방식을 공개 교리와 정합.
- **engine_core.py ENEMY_DB/ENEMY_FLEET_PRESETS**: 무인기 군집 위협(저RCS·저속·저가, 다수 동시 스폰). v16.08 연안 자폭드론 스웜(30대)·v16.11 자폭 USV 인프라 확장 — 군집 규모를 수십~수백으로, 분산 경로(다축 접근) 부여.
- **핵심 메커니즘**: 개별 요격 필요 → CIWS/SAM 채널 포화 → 탄약 급소모. 기존 채널 포화·`max_channels` 로직 재활용.
- **군집 비행(간략 모델)**: 본격 boid 알고리즘 대신, 다수 위협의 분산 스폰 좌표 + 동시 접근으로 추상화(엔진 부담 최소).
- **토글**: `enable_drone_swarm`(또는 적 편대 프리셋으로). 레이저(v17.2) 대응 효과는 v17.2서 발현.
- **검증**: 군집 규모별 탄약 소진·요격률 측정. 현실성(규모·제원) 공개 교리 대조. 기준값 `project-baseline-drone-swarm`.

## 트랙 C — 함정별 자율 교전 AI (v15.3) · 난이도 높음 · 가장 침습적

각 함정이 CEC 지령 없이 독립 판단·사격. 기함 격침 시 차순위 지휘권 인수.

- **engine_combat.py `_friendly_defense` 표적 할당**: 현재 전단 중앙 집중(CEC) 할당을, 함정별 독립 판단(각 함정이 자기 센서·사거리 내 위협을 자율 교전)으로 분기.
- **지휘권 인수**: 기함(primary) 격침 시 차순위 생존 함정이 지휘권 인수(`_primary()` 재선정 로직 확장).
- **자율화 수준**: 반자동(지령 우선)~완전자율(독립) 조정 토글/슬라이더.
- **토글**: `enable_autonomous_engagement`. **가장 침습적** → 설계 먼저 합의·코드리뷰 high·회귀 신중(CEC OFF 경로와 상호작용).
- **검증**: CEC ON/OFF × 자율 ON/OFF 4조합 회귀·효과. 기함 격침 시 지휘권 인수 단위검증. 기준값 `project-baseline-autonomous`.
