# 이지스 기동전단 통합 방어 시뮬레이터

한국 해군 이지스 기동전단의 **다층 방어 시뮬레이터**입니다. 대공·대함·대잠 위협에 대한 교전 시뮬레이션, 몬테카를로 분석, 요구조건(REQ) 판정, Excel/PNG 보고서 생성을 수행합니다.

> **현재 버전: v11.10**

---

## 주요 기능

- **시간 스텝 기반 양방향 교전 엔진** — 탐지·추적·요격을 초 단위로 시뮬레이션
- **다층 방어 모델** — SAM(SM-3/SM-2/ESSM/해궁) · CIWS(Phalanx/RAM) · 전자전 · 채프/플레어
- **대잠전(ASW)** — 함재 헬기·해상초계기(P-8A 등)의 어뢰 교전
- **몬테카를로 분석** — 표준 MC + LHS 고속 샘플링(멀티프로세싱 병렬)
- **방대한 무기 체계 DB** — 적군 65종 · 아군 방어 14종 · 함정 21종 · 편대 프리셋 등
- **작전 시나리오 라이브러리** — 교리 기반 시나리오 선택 시 편대·해역·날씨 자동 설정
- **보고서 출력** — 교전 결과를 Excel(.xlsx) · PNG로 내보내기

---

## 실행 방법

### 개발 환경에서 실행

```powershell
python launcher.py
```

### 실행 파일(exe) 빌드

```powershell
python -m PyInstaller launcher.spec --noconfirm
```

빌드 결과물은 `dist/이지스_기동전단_시뮬레이터/` 폴더에 생성됩니다.

## 파일 구조

| 파일 | 역할 |
|------|------|
| `engine.py` | 핵심 DB(적/아군/함정), 물리 모델, 탐지·교전 로직 |
| `engine_v7.py` | 시간 스텝 기반 양방향 교전 엔진 |
| `launcher.py` | PyQt6 런처 — UI, 시뮬 워커, 결과·DB·계획 탭 등 전체 앱 |
| `spec_db.py` | DB 탭 스펙시트용 상세 설명 |
| `changelog.json` | 패치 이력 |
| `launcher.spec` | PyInstaller 빌드 스펙 |
| `assets/images/` | DB 항목별 장비 사진 |

## 사용 흐름

1. 시뮬레이터 시작 화면에서 **[🚀 시뮬레이터 시작]** 클릭
2. 설정 패널에서 아군 편대·적 위협·해역·날씨 등 구성
3. **[▶ 시뮬레이션 실행]** 또는 몬테카를로 분석 실행
4. 결과 탭에서 요격률·비용·손상 등 확인
5. Excel/PNG 보고서 내보내기
