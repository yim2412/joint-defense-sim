# 종합 감사 보고서

> 메이저 블록 완료 시 수행하는 **종합 감사**(9영역)의 기록.
> 규칙 정본은 `CLAUDE.md`의 '종합 감사' 섹션. 이 파일은 **산출물**이다.
>
> **사용법**: 블록 감사를 마치면 아래 형식으로 **최신이 위로 오도록** 섹션을 추가한다.
> 각 영역은 **PASS / 발견N / 해당없음** 중 하나. 발견 항목은 그 자리에서 수정 후
> 커밋(다음 일련번호)하고 '조치'에 버전 기록. **9영역 전부 PASS 또는 해당없음**이어야
> 블록 종료 선언. 텍스트 에디터·메모장으로 바로 열람 가능.
>
> `audit_scan.py`(정적 점검 자동 스캐너)는 **첫 종합 감사 실전에서** 손으로 친 Grep을
> 추출해 만든다(추측 선제작 금지 — YAGNI).
